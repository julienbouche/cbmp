This project aims to create a map with all the informations you can imagine about the great craft brewers around the world. It is aimed to be owned by its users and this is why the project is totally open-source and new informations can be added by everyone.

The site is relying on open-source frameworks such as : OpenLayers and OpenStreetMap. The code is freely available on Github. Feel free to help this project by coding, by adding informations on places or adding new great places.

Don't forget to have a beer and drink responsibly.

Cheers!
